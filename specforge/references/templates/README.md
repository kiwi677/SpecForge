# Templates

`templates/` 保存 SpecForge 生成方案时可复用的结构模板。

## 文件说明

| 文件 | 用途 |
| --- | --- |
| `project-template.md` | 新项目入库模板 |
| `prototype-template.md` | 原型图内容检查模板 |
| `html-preview-template.html` | HTML 方案页基础骨架 |
| `acceptance-checklist.md` | 方案质量门禁和验收清单 |

## 使用原则

- 模板是起点，不是固定答案。
- 没有内容的板块不展示。
- 生成方案必须结合项目知识库、playbook 和用户输入。
- HTML 预览页必须能离线打开，不依赖外部资源。
